# 医疗影像模型管理平台 - 系统页面设计方案

## 🎯 设计概述

基于当前已实现的登录和注册功能，设计完整的系统内部页面架构，包括主页、系统配置页以及公共组件。

## 🏗️ 整体架构设计

### 页面布局结构
```
系统内部页面
├── 公共组件
│   ├── MainLayout.vue          # 主布局容器
│   ├── Sidebar.vue             # 侧边栏导航
│   ├── Header.vue              # 顶部导航栏
│   ├── Breadcrumb.vue          # 面包屑导航
│   └── Footer.vue              # 页脚信息
├── 核心页面
│   ├── Dashboard.vue           # 主页/控制台
│   ├── SystemConfig.vue        # 系统配置页
│   └── UserProfile.vue         # 用户资料页
└── 功能模块页面
    ├── UserManagement/         # 用户管理
    ├── PermissionManagement/    # 权限管理
    ├── CreditSystem/           # 积分系统
    └── ModelManagement/        # 模型管理
```

## 🎨 设计原则

### 视觉设计
- **医疗专业性**: 采用医疗行业常用的蓝色系配色，体现专业性和可信度
- **现代化界面**: 使用Element Plus组件库，确保界面美观和一致性
- **响应式设计**: 支持多种屏幕尺寸，移动端友好

### 交互设计
- **直观导航**: 清晰的菜单结构和面包屑导航
- **快速访问**: 常用功能快速入口和快捷操作
- **状态反馈**: 清晰的操作反馈和状态提示

## 🧩 公共组件设计

### 1. MainLayout.vue - 主布局容器
- **功能**: 系统页面的整体布局容器
- **特性**: 
  - 响应式侧边栏（可折叠）
  - 固定顶部导航
  - 动态内容区域
  - 面包屑导航支持

### 2. Sidebar.vue - 侧边栏导航
- **功能**: 系统主要功能模块导航
- **菜单结构**:
  ```
  📊 控制台
  👥 用户管理
  🔐 权限管理
  💰 积分系统
  ⚙️ 系统配置
  🤖 模型管理
  📊 数据统计
  ```

### 3. Header.vue - 顶部导航栏
- **功能**: 用户信息、通知、设置等快捷操作
- **内容**:
  - 系统Logo和标题
  - 用户头像和下拉菜单
  - 消息通知图标
  - 主题切换按钮
  - 全屏切换按钮

### 4. Breadcrumb.vue - 面包屑导航
- **功能**: 显示当前页面在系统中的位置
- **特性**: 动态生成，支持点击跳转

## 📱 核心页面设计

### 1. Dashboard.vue - 主页/控制台
- **功能**: 系统概览和快速入口
- **内容模块**:
  - 欢迎信息卡片
  - 系统状态概览
  - 快速操作入口
  - 最近活动记录
  - 数据统计图表
  - 快捷功能卡片

### 2. SystemConfig.vue - 系统配置页
- **功能**: 系统参数配置和管理
- **内容模块**:
  - 配置分类标签页
  - 配置项列表/表单
  - 批量操作工具栏
  - 配置搜索和筛选
  - 配置导入/导出

## 🎨 视觉设计规范

### 色彩系统
- **主色调**: #1890FF (医疗蓝)
- **辅助色**: #52C41A (成功绿), #FAAD14 (警告橙), #F5222D (错误红)
- **中性色**: #F5F5F5 (背景), #FFFFFF (卡片), #000000 (文字)
- **边框色**: #E8E8E8, #D9D9D9

### 字体系统
- **主标题**: 20px, 粗体
- **次标题**: 16px, 粗体
- **正文**: 14px, 常规
- **辅助文字**: 12px, 常规
- **说明文字**: 12px, 细体

### 间距系统
- **基础单位**: 8px
- **组件间距**: 16px, 24px
- **页面边距**: 24px, 32px
- **卡片内边距**: 16px, 24px

## 🔧 技术实现要点

### 组件通信
- 使用Pinia进行状态管理
- 组件间通过props和emit通信
- 全局状态通过store共享

### 响应式设计
- 使用CSS Grid和Flexbox布局
- 媒体查询断点: xs(<480px), sm(<768px), md(<992px), lg(<1200px), xl(≥1200px)
- 侧边栏在小屏幕下自动折叠

### 性能优化
- 路由懒加载
- 组件按需引入
- 图片懒加载
- 虚拟滚动（大数据列表）
